## Usage

### Creating a sketchbook

```sh
p5 book new sketches
cd sketches
```

### Creating a sketch in the collection

```sh
p5 sketch new my_sketch
```

### Running your sketches

```sh
p5 serve
```

### Sketch development server with hot reloading

```sh
p5 serve --dev
```
